JavaScript Objects

Object Literals

Objects are collections of key-value pairs.

// Object literal
const person = {
    firstName: "John",
    lastName: "Doe",
    age: 30,
    isStudent: false,
    greet: function() {
        console.log(`Hello, my name is ${this.firstName}`);
    }
};

// Accessing properties
console.log(person.firstName); // "John"
person.greet(); // "Hello, my name is John"

Object Methods and 'this'

Methods are functions that are properties of an object.

const car = {
    brand: "Toyota",
    model: "Corolla",
    year: 2020,
    getAge: function() {
        const currentYear = new Date().getFullYear();
        return currentYear - this.year;
    }
};

console.log(car.getAge()); // Returns the car's age

Object Destructuring (ES6)

Extract values from objects into distinct variables.

const user = {
    id: 42,
    displayName: 'jdoe',
    fullName: {
        firstName: 'John',
        lastName: 'Doe'
    }
};

// Destructuring
const { id, displayName } = user;
const { firstName, lastName } = user.fullName;

console.log(id); // 42
console.log(firstName); // "John"

Classes (ES6)

Syntactic sugar over JavaScript's prototype-based inheritance.

class Person {
    constructor(name, age) {
        this.name = name;
        this.age = age;
    }

    greet() {
        console.log(`Hello, my name is ${this.name}`);
    }
}

const person1 = new Person("Alice", 25);
person1.greet(); // "Hello, my name is Alice"
