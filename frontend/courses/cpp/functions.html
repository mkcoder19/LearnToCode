C++ Functions

Function Basics

Functions are reusable blocks of code that perform specific tasks.

// Function declaration
int addNumbers(int a, int b);

// Function definition
int addNumbers(int a, int b) {
    return a + b;
}

// Function call
int result = addNumbers(5, 3); // Returns 8
Component Description Example
Return Type Data type of returned value int
Function Name Identifier for the function addNumbers
Parameters Input values for the function (int a, int b)
Function Body Code block with statements { return a + b; }

Parameter Passing

C++ supports different ways to pass parameters to functions:

// Pass by value (creates copy)
void modifyValue(int x) {
    x = x * 2;
}

// Pass by reference (modifies original)
void modifyReference(int &x) {
    x = x * 2;
}

// Pass by const reference (efficient for large objects)
void printLargeObject(const BigObject &obj) {
    // Can read but not modify obj
}

int num = 5;
modifyValue(num); // num remains 5
modifyReference(num); // num becomes 10

Function Overloading

C++ allows multiple functions with same name but different parameters.

// Overloaded functions
int add(int a, int b) {
    return a + b;
}

double add(double a, double b) {
    return a + b;
}

string add(string a, string b) {
    return a + b;
}

add(5, 3); // Calls int version
add(2.5, 3.7); // Calls double version
add("Hello", "World"); // Calls string version

Lambda Functions (C++11)

Anonymous functions that can be defined inline.

// Simple lambda
auto greet = []() {
    cout << "Hello World!";
};
greet();

// Lambda with parameters
auto add = [](int a, int b) {
    return a + b;
};
int sum = add(5, 3);

// Capturing variables
int multiplier = 3;
auto times = [multiplier](int x) {
    return x * multiplier;
};
int result = times(5); // 15
