C++ Pointers

Pointer Basics

A pointer is a variable that stores the memory address of another variable.

int num = 10;
int *ptr = # // Pointer declaration and initialization

// Accessing value using pointer
cout << "Value of num: " << *ptr; // 10
cout << "Address of num: " << ptr; // Address of num

Pointer Arithmetic

Pointer arithmetic allows you to navigate through memory addresses.

int arr[] = {10, 20, 30};
int *ptr = arr; // Points to the first element

cout << *ptr; // 10
ptr++; // Move to the next integer
cout << *ptr; // 20

Dynamic Memory Allocation

Using pointers, you can allocate memory dynamically using new and deallocate with delete.

int *dynamicArray = new int[5]; // Allocate array of 5 integers

// Initialize and access
for (int i = 0; i < 5; i++) {
    dynamicArray[i] = i * 10;
}

delete[] dynamicArray; // Deallocate memory

Smart Pointers (C++11)

Smart pointers manage memory automatically and help prevent memory leaks.

#include 

std::unique_ptr ptr1(new int(10)); // Unique pointer
std::shared_ptr ptr2 = std::make_shared(20); // Shared pointer

cout << *ptr1; // 10
cout << *ptr2; // 20
