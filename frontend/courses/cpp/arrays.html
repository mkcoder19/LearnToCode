C++ Arrays and Vectors

C-Style Arrays

Traditional fixed-size arrays inherited from C:

// Array declaration and initialization
int numbers[5]; // Uninitialized array of 5 integers
int primes[] = {2, 3, 5, 7, 11}; // Initialized array

// Accessing elements
numbers[0] = 10; // First element
int firstPrime = primes[0]; // 2

// Size calculation
int size = sizeof(primes) / sizeof(primes[0]); // 5

// Looping through array
for (int i = 0; i < size; i++) {
    cout << primes[i] << " ";
}

std::array (C++11)

Safer, more modern alternative to C-style arrays:

#include 

// Declaration
std::array numbers;

// Initialization
std::array primes = {2, 3, 5, 7, 11};

// Accessing elements
numbers[0] = 10;
int firstPrime = primes.at(0); // Bounds-checked access

// Useful methods
int size = primes.size(); // 5
bool empty = primes.empty(); // false
primes.fill(0); // Fill all elements with 0

Multidimensional Arrays

// 2D C-style array
int matrix[3][3] = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

// Accessing elements
int center = matrix[1][1]; // 5

// 2D std::array
std::array, 3> modernMatrix = {{
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
}};

// Range-based for loop (C++11)
for (const auto &row : modernMatrix) {
    for (int num : row) {
        cout << num << " ";
    }
    cout << endl;
}

Array Limitations and Alternatives

For dynamic sizing, consider std::vector:

#include 

// Vector declaration
std::vector dynamicArray;

// Adding elements
dynamicArray.push_back(10);
dynamicArray.push_back(20);
dynamicArray.push_back(30);

// Accessing elements
int first = dynamicArray[0]; // 10
int last = dynamicArray.back(); // 30

// Size management
int size = dynamicArray.size(); // 3
dynamicArray.resize(5); // Now has 5 elements
dynamicArray.shrink_to_fit(); // Reduce capacity
