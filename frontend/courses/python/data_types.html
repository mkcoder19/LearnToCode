Python Data Types

Basic Data Types

Python has several built-in data types:

Numbers

# Integers
age = 25
temperature = -10

# Floating point numbers
price = 19.99
pi = 3.14159

# Complex numbers
complex_num = 3 + 4j

Strings

name = "Alice"
message = 'Hello, World!'
multiline = """This is a
multi-line string"""

Booleans

is_active = True
is_admin = False

Type Conversion

You can convert between different types:

# String to integer
num_str = "123"
num_int = int(num_str)

# Integer to float
int_num = 5
float_num = float(int_num)

# Float to string
pi = 3.14159
pi_str = str(pi)
