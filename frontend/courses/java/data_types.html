Java Data Types

Primitive Data Types

Java has 8 primitive data types that form the building blocks of Java programs:

Type Size Range Default Example
byte 8-bit -128 to 127 0 byte b = 100;
short 16-bit -32,768 to 32,767 0 short s = 10000;
int 32-bit -231 to 231-1 0 int i = 100000;
long 64-bit -263 to 263-1 0L long l = 10000000000L;
float 32-bit ±1.4E-45 to ±3.4E+38 0.0f float f = 3.14f;
double 64-bit ±4.9E-324 to ±1.7E+308 0.0d double d = 3.1415926535;
char 16-bit '\u0000' to '\uffff' '\u0000' char c = 'A';
boolean 1-bit true/false false boolean flag = true;

Practical Examples

// Temperature measurement
byte currentTemp = 25;  // -128 to 127 range sufficient

// Student grade calculation
float averageGrade = 89.5f;

// Large calculations
double preciseValue = 2.718281828459045;

// Character handling
char gradeLetter = 'A';

// Status flags
boolean isPassing = true;

Reference Data Types

Java also has reference types that store references to objects:

// String - sequence of characters
String name = "John Doe";

// Arrays - collections of elements
int[] numbers = {1, 2, 3, 4, 5};

// Classes - custom types
class Student {
    String name;
    int age;
}
Student s1 = new Student();

Type Conversion Examples

// Widening conversion (automatic)
int myInt = 9;
double myDouble = myInt;  // Automatic: int → double

// Narrowing conversion (manual)
double myDouble = 9.78;
int myInt = (int) myDouble;  // Manual: double → int

// String conversion
String str = "123";
int num = Integer.parseInt(str);
