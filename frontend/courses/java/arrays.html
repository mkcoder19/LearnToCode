Java Arrays

Array Basics

Arrays are fixed-size containers that hold multiple values of the same type.

// Array declaration and initialization
int[] numbers = new int[5]; // Array of 5 integers
String[] names = {"Alice", "Bob", "Charlie"}; // Initialized array

// Accessing elements
numbers[0] = 10; // First element
int firstNumber = numbers[0];
String secondName = names[1]; // "Bob"
Feature Description Example
Declaration Specifying type and name int[] numbers;
Instantiation Creating with new keyword numbers = new int[5];
Initialization Assigning values int[] nums = {1,2,3};
Length Fixed size property int size = numbers.length;

Multidimensional Arrays

Arrays can have multiple dimensions (arrays of arrays).

// 2D array (matrix)
int[][] matrix = new int[3][3]; // 3x3 grid
matrix[0][0] = 1; // First row, first column

// Initializing 2D array
int[][] grid = {
    {1, 2, 3},
    {4, 5, 6},
    {7, 8, 9}
};

// Accessing elements
int center = grid[1][1]; // 5

Array Operations

// Looping through array
for (int i = 0; i < numbers.length; i++) {
    System.out.println(numbers[i]);
}

// Enhanced for loop
for (String name : names) {
    System.out.println(name);
}

// Array copy
int[] copy = Arrays.copyOf(numbers, numbers.length);

// Sorting
Arrays.sort(numbers);
