Linked Lists in Data Structures

Theory

Linked lists are a linear data structure where elements are stored in nodes, each pointing to the next node.

Key Characteristics:

  • Dynamic size
  • O(n) access time for elements
  • Efficient insertions/deletions

Types and Java Implementation:

  • Singly Linked: Nodes with next pointer only
  • Doubly Linked: Nodes with next and prev pointers
  • Circular: Last node points back to first
// Node class for Singly Linked List
class Node {
    int data;
    Node next;
    Node(int data) { this.data = data; }
}

// Doubly Linked List Node
class DNode {
    int data;
    DNode next, prev;
    DNode(int data) { this.data = data; }
}

// Basic LinkedList operations
class LinkedList {
    Node head;
    
    void insert(int data) {
        Node newNode = new Node(data);
        newNode.next = head;
        head = newNode;
    }
    
    void printList() {
        Node current = head;
        while (current != null) {
            System.out.print(current.data + " ");
            current = current.next;
        }
    }
}
