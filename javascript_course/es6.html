Modern JavaScript (ES6+)

Block-Scoped Declarations

ES6 introduced let and const for better variable scoping:

// let allows reassignment
let counter = 0;
counter = 1; // Valid

// const prevents reassignment
const MAX_SIZE = 100;
// MAX_SIZE = 200; // Error!

// Block scoping
if (true) {
    let localVar = 'only visible in this block';
    const LOCAL_CONST = 'also block-scoped';
}

Arrow Functions

Concise syntax and lexical this binding:

// Traditional function
function add(a, b) {
    return a + b;
}

// Arrow function equivalent
const add = (a, b) => a + b;

// With single parameter
const square = x => x * x;

// With body block
const greet = name => {
    const message = `Hello, ${name}!`;
    return message;
};

Template Literals

Multi-line strings and expression interpolation:

const name = 'Alice';
const age = 25;

// String interpolation
const message = `Hello ${name}, you are ${age} years old!`;

// Multi-line strings
const html = `


    
${name}

    
Age: ${age}



`;

Destructuring Assignment

Unpack values from arrays or objects:

// Array destructuring
const [first, second] = [1, 2, 3];

// Object destructuring
const person = { name: 'Bob', age: 30 };
const { name, age } = person;

// Function parameter destructuring
function greet({ name, age }) {
    return `Hello ${name}, age ${age}`;
}

Spread and Rest Operators

// Spread operator (...)
const numbers = [1, 2, 3];
const newNumbers = [...numbers, 4, 5]; // [1, 2, 3, 4, 5]

// Rest parameters
function sum(...nums) {
    return nums.reduce((total, num) => total + num, 0);
}
sum(1, 2, 3); // 6
