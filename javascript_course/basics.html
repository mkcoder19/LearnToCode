JavaScript Basics

Variables and Data Types

JavaScript has dynamic typing and several ways to declare variables:

// Variable declarations
var oldVariable = "Using var (function-scoped)";
let mutableVariable = "Using let (block-scoped)";
const constantVariable = "Using const (block-scoped, immutable)";

// Data types
let number = 42;               // Number
let text = "Hello";            // String
let isTrue = true;             // Boolean
let nothing = null;            // Null
let notDefined;                // Undefined
let person = {                 // Object
    name: "Alice",
    age: 25
};
let colors = ["red", "green"]; // Array
Type Description Example
Number Numeric values let age = 25;
String Text data let name = "Alice";
Boolean True/False let isActive = true;
Object Key-value pairs let user = { name: "Bob" };
Array Ordered lists let nums = [1, 2, 3];
Null/Undefined No value let empty = null;

Type Conversion and Coercion

JavaScript automatically converts types in certain contexts:

// Explicit conversion
let num = "42";
let realNum = Number(num); // Convert to number

// Implicit coercion
let result1 = "5" + 2;    // "52" (string concatenation)
let result2 = "5" - 2;    // 3 (numeric subtraction)

// Truthy/Falsy values
if ("hello") { /* executes */ }
if (0) { /* doesn't execute */ }

Basic Output and Script Usage

Ways to run JavaScript and display output:

// Console output
console.log("Hello Console!");

// Browser alert
