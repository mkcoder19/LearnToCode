Java Operators

Arithmetic Operators

Operator Description Example
+ Addition int sum = 5 + 3; // 8
- Subtraction int diff = 5 - 3; // 2
* Multiplication int product = 5 * 3; // 15
/ Division int quotient = 5 / 3; // 1
% Modulus int remainder = 5 % 3; // 2

Practical Examples

// Calculate area of rectangle
int length = 10;
int width = 5;
int area = length * width; // 50

// Convert minutes to hours and minutes
int totalMinutes = 125;
int hours = totalMinutes / 60; // 2
int minutes = totalMinutes % 60; // 5

Relational Operators

Operator Description Example
== Equal to 5 == 3 // false
!= Not equal 5 != 3 // true
> Greater than 5 > 3 // true
< Less than 5 < 3 // false
>= Greater than or equal 5 >= 5 // true
<= Less than or equal 5 <= 3 // false
