Java Methods

Method Basics

Methods are blocks of code that perform specific tasks and can be reused throughout your program.

// Method definition
public static int addNumbers(int a, int b) {
    return a + b;
}

// Method call
int sum = addNumbers(5, 3); // Returns 8
Component Description Example
Access Modifier Determines visibility (public, private, protected) public
Return Type Data type of the returned value (void if none) int
Method Name Identifier for the method addNumbers
Parameters Input values for the method (int a, int b)
Method Body Code block with statements { return a + b; }

Method Overloading

Java allows multiple methods with same name but different parameters.

// Overloaded methods
public static int add(int a, int b) {
    return a + b;
}

public static double add(double a, double b) {
    return a + b;
}

public static int add(int a, int b, int c) {
    return a + b + c;
}

Recursion

A method that calls itself is called recursive.

// Factorial using recursion
public static int factorial(int n) {
    if (n == 0 || n == 1) {
        return 1;
    }
    return n * factorial(n - 1);
}

// Calculate 5! (120)
int result = factorial(5);
