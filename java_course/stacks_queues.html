Stacks & Queues in Data Structures

Theory

Stacks and Queues are fundamental linear data structures with specific insertion and removal patterns.

Stack (LIFO):

  • Last In First Out principle
  • Operations: push(), pop(), peek()
  • Used in function calls, undo operations

Queue (FIFO):

  • First In First Out principle
  • Operations: enqueue(), dequeue(), peek()
  • Used in scheduling, buffering

  • Stack Implementations: Array, LinkedList
  • Queue Types: Simple, Circular, Priority
  • Deque: Double-ended queue
// Stack using Array
class ArrayStack {
    int[] arr;
    int top;
    ArrayStack(int size) {
        arr = new int[size];
        top = -1;
    }
    void push(int val) { arr[++top] = val; }
    int pop() { return arr[top--]; }
}

// Queue using LinkedList
class ListQueue {
    Node front, rear;
    class Node {
        int data;
        Node next;
        Node(int data) { this.data = data; }
    }
    void enqueue(int val) {
        Node newNode = new Node(val);
        if (rear == null) {
            front = rear = newNode;
        } else {
            rear.next = newNode;
            rear = newNode;
        }
    }
    int dequeue() {
        if (front == null) return -1;
        int val = front.data;
        front = front.next;
        if (front == null) rear = null;
        return val;
    }
}
