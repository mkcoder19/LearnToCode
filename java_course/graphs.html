Graphs in Data Structures

Theory

Graphs are non-linear data structures consisting of nodes (vertices) connected by edges.

Key Characteristics:

  • Vertices (nodes) and edges (connections)
  • Can be directed or undirected
  • Used in social networks, maps, routing

Types and Java Implementation:

  • Adjacency List: Efficient for sparse graphs
  • Adjacency Matrix: Suitable for dense graphs
  • Edge List: Simple but less efficient

Java Implementations:

import java.util.*;

// Adjacency List Implementation
class Graph {
    private Map> adjList;
    
    public Graph() {
        adjList = new HashMap<>();
    }
    
    public void addEdge(int u, int v) {
        adjList.computeIfAbsent(u, k -> new ArrayList<>()).add(v);
        adjList.computeIfAbsent(v, k -> new ArrayList<>()).add(u); // For undirected graph
    }

    public void printGraph() {
        for (Map.Entry> entry : adjList.entrySet()) {
            System.out.print(entry.getKey() + ": ");
            for (int neighbor : entry.getValue()) {
                System.out.print(neighbor + " ");
            }
            System.out.println();
        }
    }
}

// Adjacency Matrix Implementation
class AdjacencyMatrixGraph {
    private int[][] matrix;
    private int size;

    public AdjacencyMatrixGraph(int size) {
        this.size = size;
        matrix = new int[size][size];
    }

    public void addEdge(int u, int v) {
        matrix[u][v] = 1; // For directed graph
        matrix[v][u] = 1; // For undirected graph
    }

    public void printMatrix() {
        for (int i = 0; i < size; i++) {
            for (int j = 0; j < size; j++) {
                System.out.print(matrix[i][j] + " ");
            }
            System.out.println();
        }
    }
}
