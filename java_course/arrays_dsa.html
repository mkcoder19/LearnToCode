Arrays in Data Structures

Theory

Arrays are contiguous blocks of memory used to store multiple items of the same type.

Key Characteristics:

  • Fixed size (static)
  • O(1) access time for any element
  • Memory efficient for sequential data

Types and Java Implementation:

  • 1D Arrays: Single row of elements
  • 2D Arrays: Matrix structure
  • Dynamic Arrays: ArrayList class
// 1D Array
int[] arr = {10, 20, 30};
System.out.println(arr[1]); // 20

// 2D Array 
int[][] matrix = {{1,2}, {3,4}};
System.out.println(matrix[0][1]); // 2

// Dynamic Array (ArrayList)
ArrayList names = new ArrayList<>();
names.add("Alice");
names.add("Bob");
