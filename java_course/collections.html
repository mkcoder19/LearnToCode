Java Collections Framework

Core Collection Interfaces

The Collections Framework provides various interfaces and implementations for storing and processing groups of objects.

Interface Description Common Implementations
List Ordered collection (allows duplicates) ArrayList, LinkedList
Set Collection with no duplicates HashSet, TreeSet
Map Key-value pairs HashMap, TreeMap
Queue First-in-first-out (FIFO) LinkedList, PriorityQueue

List Examples

// ArrayList example
List names = new ArrayList<>();
names.add("Alice");
names.add("Bob");
names.add(0, "Charlie"); // Insert at index 0

// LinkedList example
List numbers = new LinkedList<>();
numbers.add(10);
numbers.add(20);
numbers.get(1); // 20

Set and Map Examples

// HashSet example
Set uniqueNames = new HashSet<>();
uniqueNames.add("Alice");
uniqueNames.add("Bob");
uniqueNames.add("Alice"); // Duplicate ignored

// HashMap example
Map ageMap = new HashMap<>();
ageMap.put("Alice", 25);
ageMap.put("Bob", 30);
int aliceAge = ageMap.get("Alice"); // 25

Common Operations

// Iterating through collections
for (String name : names) {
    System.out.println(name);
}

// Sorting a list
Collections.sort(names);

// Checking contains
if (uniqueNames.contains("Alice")) {
    System.out.println("Alice is present");
}

// Map iteration
for (Map.Entry entry : ageMap.entrySet()) {
    System.out.println(entry.getKey() + ": " + entry.getValue());
}
