Python Arrays

Introduction to Arrays

In Python, arrays are not a built-in data type, but you can use lists to achieve similar functionality.

# Creating an array (list)
numbers = [1, 2, 3, 4, 5]

# Accessing elements
print(numbers[0])  # 1

# Modifying elements
numbers[1] = 10
print(numbers)  # [1, 10, 3, 4, 5]
