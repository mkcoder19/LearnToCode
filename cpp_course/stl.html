C++ Standard Template Library (STL)

Containers

The STL provides various container classes for storing data.

Container Description Example
vector Dynamic array vector v = {1, 2, 3};
list Doubly-linked list list names;
map Key-value pairs map ages;
set Unique elements set uniqueNumbers;
queue FIFO structure queue tasks;
stack LIFO structure stack characters;

Algorithms

The STL includes powerful algorithms that work with containers.

vector numbers = {5, 2, 8, 1, 9};

// Sorting
sort(numbers.begin(), numbers.end());

// Searching
auto it = find(numbers.begin(), numbers.end(), 8);

// Reversing
reverse(numbers.begin(), numbers.end());

// Counting
int count = count_if(numbers.begin(), numbers.end(), 
                    [](int x){ return x > 5; });

Iterators

Iterators provide a way to access elements in containers.

vector fruits = {"Apple", "Banana", "Orange"};

// Using iterators
for (auto it = fruits.begin(); it != fruits.end(); ++it) {
    cout << *it << endl;
}

// Range-based for loop (C++11)
for (const auto& fruit : fruits) {
    cout << fruit << endl;
}

Function Objects and Lambdas

STL algorithms can be customized with functions and lambdas.

// Function object
struct Square {
    int operator()(int x) const { return x * x; }
};

vector nums = {1, 2, 3, 4, 5};

// Transform with function object
transform(nums.begin(), nums.end(), nums.begin(), Square());

// Lambda expression
sort(nums.begin(), nums.end(), 
     [](int a, int b){ return a > b; }); // Descending sort
