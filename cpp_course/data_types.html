C++ Data Types

Fundamental Data Types

C++ provides a rich set of built-in data types:

Type Size (bytes) Description Example
int 4 Integer numbers int age = 25;
float 4 Single-precision floating point float price = 9.99f;
double 8 Double-precision floating point double pi = 3.1415926535;
char 1 Single character char grade = 'A';
bool 1 Boolean (true/false) bool is_valid = true;
void N/A No type void displayMessage();

Type Modifiers

C++ allows type modifiers to change the meaning of basic types:

// Signed and unsigned
signed int negative = -10;  // Can be negative
unsigned int positive = 10; // Only positive

// Short and long
short smallNumber = 100;
long bigNumber = 1000000L;
long long veryBigNumber = 10000000000LL;

// Size demonstration
cout << "Size of int: " << sizeof(int) << " bytes" << endl;
cout << "Size of long: " << sizeof(long) << " bytes" << endl;

Type Conversion

C++ supports both implicit and explicit type conversion:

// Implicit conversion
int num_int = 10;
double num_double = num_int; // Automatic conversion

// Explicit conversion (C-style)
double pi = 3.14159;
int approx_pi = (int)pi; // Truncates to 3

// C++ style casting
int quotient = static_cast(10.5 / 2.5); // 4
